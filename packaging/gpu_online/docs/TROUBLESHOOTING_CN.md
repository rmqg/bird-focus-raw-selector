# GPU 便携包故障排查

## 1) 提示未检测到 NVIDIA 环境

- 本包是 GPU 专用包。
- 请先确认 `nvidia-smi` 可用。
- 若当前机器无 NVIDIA 环境，请使用 CPU 便携包。

## 2) 首次运行很慢

- 首次会创建运行时虚拟环境并安装 GPU 依赖。
- 这是一次性开销，后续会复用 `.runtime_venv`。

## 3) 模型下载失败

- 检查网络后重试。
- 可手动把 `yolov8s-seg.pt` 放到程序同目录，再重新运行。

## 4) 依赖安装报空间不足

- 建议把程序放在剩余空间更大的磁盘分区（如 `E:\` / `D:\`）。
- 程序会将安装缓存写入包目录下，不依赖系统盘临时目录。

## 5) 结果异常

- 先查看日志里的 `failure_reason` 字段。
- 如需调参，请附带日志和样例 RAW 再做下一轮优化。
