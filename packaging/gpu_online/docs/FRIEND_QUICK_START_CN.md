# GPU 便携包快速使用说明

## 1. 本包特点

- 仅支持 GPU 模式（不含 CPU 启动入口）。
- 已内置 Python 运行环境，无需单独安装 Python。
- 首次运行会自动安装 GPU 依赖并检查/下载模型。

## 2. 使用顺序

1. 双击 `Run_DryRun_Fast_GPU.bat`（先预览，不复制）。
2. 确认结果后，双击 `Run_Copy_Fast_GPU.bat`（正式复制）。

## 3. 运行时输入

- 源目录：RAW 根目录（会递归扫描）。
- 输出目录：筛选命中的 RAW 复制到这里（仅复制，不改原文件）。
- 日志路径：保存每张图的结构化结果（CSV）。

## 4. 注意事项

- 必须有可用 NVIDIA 环境（`nvidia-smi` 可用）。
- 首次安装依赖会下载较大文件，建议保持稳定网络。
- 自动跳过 `selected_birds_in_focus*` 与 `raw*` 子目录。
